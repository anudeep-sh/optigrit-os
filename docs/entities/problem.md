# Problem

## Definition

A Problem is an individual challenge presented to a learner within a Contest or practice environment.

Problems evaluate knowledge, reasoning or implementation skills.

---

## Problem Types

- Multiple Choice
- Coding
- Descriptive
- Numerical
- File Upload
- Case Analysis

---

## Owns

- Statement
- Constraints
- Difficulty
- Test Cases
- Evaluation Rules
- Hints
- Solution

---

Problem

## Invariants

Problems remain versioned.

Published Problems cannot change evaluation behaviour.

Solutions remain hidden until release conditions are met.
