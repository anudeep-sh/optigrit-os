# Learning Asset

## Definition

A Learning Asset is the smallest reusable educational component.

Everything students interact with should eventually become a Learning Asset.

---

## Examples

Video

PDF

Presentation

Assignment Template

Contest

Coding Exercise

Interactive Lab

Simulation

Reading Material

External Resource

AI Conversation

---

## Why Atomic Assets?

Atomic assets enable reuse.

Instead of copying videos between courses,

multiple Courses reference the same Learning Asset.

Improvements automatically benefit every Course.

## Ownership

Learning Assets belong to Knowledge.

Courses organize Learning Assets.

Teaching delivers Learning Assets.

Learning records interaction.
