# Contest

## Definition

A Contest is a time-bound competitive learning activity designed to evaluate learners under predefined rules.

Contests encourage participation, practice and benchmarking.

---

## Contest Types

- Coding Contest
- Quiz Contest
- Hackathon
- Challenge
- Case Study
- Innovation Competition

---

## Owns

- Title
- Rules
- Duration
- Start Time
- End Time
- Ranking Rules
- Visibility
- Prize Metadata

## Invariants

Contest rules are immutable after publication.

Rankings are reproducible.

Submission timestamps determine eligibility.
