# Competency

## Definition

A Competency represents a demonstrable skill or ability acquired through learning.

Unlike Learning Outcomes, which define educational intent, Competencies represent verified learner capability.

---

## Purpose

Competencies provide a long-term view of learner development beyond individual Courses or Semesters.

---

## Owns

- Name
- Description
- Category
- Proficiency Scale
- Verification Rules

---

## Relationships

Learning Outcome

---

## Invariants

Competencies are accumulated over time.

Competencies are never reset by semester changes.

Multiple Assessments may contribute to the same Competency.
