# Question Bank

## Definition

A Question Bank is a managed repository of reusable assessment questions.

Questions are independent of individual Assessments and can be reused across multiple evaluations.

---

## Owns

- Questions
- Categories
- Difficulty Levels
- Learning Outcomes
- Tags
- Version History

---

## Never Owns

Assessment Attempts.

Student Scores.

Results.

---

## Relationships

Question Bank

↓

Questions

↓

Assessment

---

## Invariants

Questions remain reusable.

Question revisions create new versions.

Published Assessments preserve the question version used during publication.
