# Progress

## Definition

Progress measures an individual learner's journey.

Progress is not attached to a Course.

It belongs to the learner.

---

## Why?

Students revisit concepts.

Repeat semesters.

Practice independently.

Study outside curriculum.

Learning is continuous.

Therefore,

Progress must remain independent of individual teaching instances.

---

## Components

Course Progress

Skill Progress

Competency Progress

Assessment Progress

Attendance Trend

Practice Activity

Learning Time

---

## Design Principle

Progress should never decrease because a Course changes.

Learning history must remain permanent.
