# Subject

## Definition

A Subject defines the academic curriculum.

It answers:

"What should students learn?"

A Subject does not define how learning happens.

That responsibility belongs to Courses.

---

## Examples

Data Structures

Operating Systems

Digital Logic

Machine Learning

Physics

Mathematics

---

## Owns

- Subject Code
- Subject Name
- Credits
- Learning Objectives
- Curriculum Metadata

---

## Never Owns

Videos

Assignments

Assessments

Attendance

Faculty

Learning Assets

These belong elsewhere.

---

## Relationship

Subject

↓

Courses

↓

Learning Assets

Multiple Courses may exist for the same Subject.

Each Course represents a different teaching implementation.

---

## Philosophy

Curriculum changes slowly.

Teaching improves continuously.

Separating Subjects from Courses allows educational quality to evolve without modifying institutional curriculum.
