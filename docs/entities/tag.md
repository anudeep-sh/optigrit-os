# Tag

## Definition

A Tag is a reusable classification label used to organize and discover information across the platform.

Tags improve searchability without changing business relationships.

---

## Purpose

Tags provide flexible categorization for educational content, assessments, resources and discussions.

---

## Owns

- Name
- Description
- Color
- Category
- Status

---

## Relationships

Tags may be associated with:

- Courses
- Learning Assets
- Resources
- Questions
- Discussions
- Tasks

---

## Invariants

Tags do not define hierarchy.

Tags are reusable across domains.

Deleting a Tag removes only the association, not the tagged entity.
