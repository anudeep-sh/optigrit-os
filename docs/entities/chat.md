# Chat

## Definition

Chat enables real-time communication between users.

Unlike Discussions, Chats are optimized for fast conversations rather than long-term knowledge sharing.

---

## Chat Types

- Direct Message
- Group Chat
- Course Chat
- Batch Chat
- Announcement Channel

---

## Owns

- Participants
- Messages
- Attachments
- Read Status
- Delivery Status

---

## Never Owns

- Academic Knowledge
- Learning Content
- Course Structure

---

## Relationships

Chat

↓

Messages

↓

## Invariants

Messages are ordered by time.

Message edits are auditable.

Deleted messages follow institutional retention policies.
