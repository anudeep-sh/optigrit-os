# Resource

## Definition

A Resource is any supplementary material that supports teaching or learning but is not itself a structured learning activity.

Resources provide additional context, reference material or supporting information.

---

## Resource Types

- PDF
- Presentation
- Image
- Audio
- Video
- External Link
- Research Paper
- Documentation
- Dataset
- Source Code
- Template

## Invariants

Resources remain independently reusable.

Replacing a Resource creates a new version.

External Resources should preserve their original source.
