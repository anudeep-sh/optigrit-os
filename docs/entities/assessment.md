# Assessment

## Definition

An Assessment measures a learner's understanding of one or more learning outcomes.

Assessments may be formative or summative.

They may be conducted online, offline or in hybrid environments.

---

## Assessment Types

- Quiz
- Assignment
- Coding Challenge
- Practical
- Viva
- Examination
- Project
- Survey

---

## Owns

- Title
- Description
- Instructions
- Questions
- Rubric
- Evaluation Rules
- Submission Policy
- Availability
- Attempts

---

## Never Owns

- Student Progress
- Course
- Results Analytics

Assessments reference these concepts but do not own them.

## Invariants

Every Assessment belongs to one Course.

Every Submission belongs to one Assessment.

Evaluation history is immutable.

Results remain auditable.
