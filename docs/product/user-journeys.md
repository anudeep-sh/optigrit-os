# User Journeys

## Purpose

User Journeys describe end-to-end business workflows.

Individual screens should never be designed in isolation from these journeys.

## Principles

Journeys should minimize context switching.

Every major step should have a clear next action.

AI should support users throughout the journey without interrupting it.
