# Data Philosophy

Data is the institutional memory of OptiGrit.

Every decision regarding storage should preserve educational history.

---

## Principles

Data should rarely be deleted.

Data should be versioned.

Data should remain explainable.

Historical context should never be lost.

---

## Immutable History

Educational history matters.

Changing today's Course should not rewrite last year's teaching.

Changing a Rubric should not modify previous evaluations.

Changing curriculum should not invalidate historical analytics.

History should represent reality.

---

## Source Of Truth

Every business concept has one source of truth.

Course

Knowledge

Attendance

Teaching

Results

Assessment

Progress

Learning

Analytics never become the source of truth.

Caches never become the source of truth.

Search indexes never become the source of truth.

---

## Time

Time is first-class.

Every significant business object should answer:

Created When

Updated When

Effective From

Effective Until

Version

Educational systems are temporal systems.

Ignoring time leads to incorrect analytics.
