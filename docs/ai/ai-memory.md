# AI Memory

## Purpose

AI Memory enables contextual assistance across the platform while respecting privacy and institutional boundaries.

---

## Memory Levels

### Session Memory

Available only during the current interaction.

---

### User Memory

Preferences specific to an individual user.

Examples

- Preferred teaching style
- Preferred language
- Formatting preferences

---

### Institutional Memory

Shared institutional knowledge.

Examples

- Policies
- Approved terminology
- Curriculum
- Course standards

---

### Platform Knowledge

General educational knowledge available to all institutions.

---

## Rules

Institutional Memory never leaks across institutions.

Users can review and modify personal preferences.

Session Memory expires automatically.
