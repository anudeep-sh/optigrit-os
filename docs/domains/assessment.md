# Assessment Domain

## Purpose

Assessment measures learning outcomes.

Teaching creates understanding.

Assessment validates understanding.

Assessment should improve learning rather than merely assign grades.

---

# Core Concepts

- Assignment
- Quiz
- Coding Challenge
- Examination
- Practical Evaluation
- Viva
- Rubric
- Submission
- Evaluation
- Result

---

# Responsibilities

Create assessments.

Collect submissions.

Evaluate submissions.

Generate results.

Provide feedback.

Support multiple evaluation methods.

---

# Assessment Principles

Assessment should encourage learning.

Feedback should be timely.

Evaluation criteria should be transparent.

Assessment should support practical skills in addition to theoretical knowledge.

---

# Invariants

Every Submission belongs to one Assessment.

An Assessment may have multiple attempts.

Results are immutable once officially published unless explicitly revised.

Evaluation history should always be preserved.
