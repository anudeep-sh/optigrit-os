# Knowledge Domain

## Purpose

Knowledge is the intellectual capital of an institution.

It represents reusable educational material independent of time, semester or faculty.

Knowledge survives across academic years.

---

# Core Objects

Subject

Course

Learning Asset

Question Bank

Resource Collection

Rubric

Reference Material

---

# Course Philosophy

A Course is not tied to:

Semester

Faculty

Batch

Section

Academic Calendar

Instead, a Course is a reusable learning experience.

The same Course can support multiple subjects across multiple semesters.

---

# Learning Assets

Learning Assets are atomic.

Examples include:

Video

Document

Presentation

External Link

Assignment Template

Contest Template

Interactive Lab

AI Conversation

Every Learning Asset should be independently reusable.

---

# Ownership Rules

Knowledge owns educational content.

Planning references content.

Teaching delivers content.

Learning consumes content.

Assessment evaluates content.

Analytics measures content effectiveness.

---

# Why This Exists

Educational content improves over time.

It should never need to be recreated because a semester changes.

Knowledge should accumulate.

Not restart.
