# Learning Domain

## Purpose

Learning represents the student's educational journey.

Unlike Teaching, which is instructor-centric, Learning is student-centric.

Every interaction between a learner and educational content belongs to this domain.

---

# Core Concepts

- Enrollment
- Learning Session
- Progress
- Completion
- Practice
- Revision
- Notes
- Bookmarks
- Certificates

---

# Responsibilities

Track learner progress.

Track engagement.

Record completed learning activities.

Support self-paced learning.

Support revision.

Support practice.

Support personalized recommendations.

---

# Progress Philosophy

Progress belongs to learners.

Not Courses.

Not Semesters.

Not Faculty.

Every learner progresses differently.

Progress is therefore a personal attribute.

---

# Invariants

Learning never modifies educational content.

Learning references Courses.

Learning references Learning Assets.

Learning owns learner state.

---

# Future Evolution

Adaptive learning.

Personalized revision plans.

AI learning assistant.

Learning path optimization.
