# Communication Domain

## Purpose

Communication connects participants in the educational process.

The objective is not messaging.

The objective is coordination.

---

# Core Concepts

- Announcement
- Chat
- Discussion
- Notification
- Mention
- Reaction
- Broadcast

---

# Communication Types

## Announcements

One-way institutional communication.

---

## Chat

Fast conversational messaging.

---

## Discussion

Persistent academic conversations.

Discussions belong to learning.

Chats belong to communication.

---

## Notifications

Action-oriented reminders.

Notifications should require attention only when necessary.

---

# Design Principles

Communication should always preserve context.

Messages should reference:

Course

Lecture

Assessment

Task

Student

Faculty

Whenever possible.

Context reduces confusion.
