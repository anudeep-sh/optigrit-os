# Versioning

## Purpose

Versioning preserves history while allowing continuous improvement.

Historical academic records must remain reproducible.

---

## Versioned Objects

- Courses
- Learning Assets
- Assessments
- Rubrics
- AI Prompt Templates
- Policies

---

## Principles

Published versions become immutable.

Corrections produce a new version rather than modifying history.

References should always indicate the version used.

---

## Examples

A learner enrolled in Course v2.1 should continue to see that version even if Course v3.0 is later published.

Assessment results should reference the exact assessment version that was attempted.

---

## Benefits

Reproducible learning experiences.

Accurate historical reporting.

Safe content evolution.
