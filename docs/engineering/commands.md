# Commands

## Purpose

Commands express an intention to perform a business action.

Unlike Events, Commands request work to be performed.

A Command may succeed or fail.

If successful, one or more Domain Events are emitted.

## Rules

Commands use imperative language.

Examples:

StartLecture

PublishCourse

GenerateAnalytics

Never:

LectureStarted

CoursePublished

Those are Events.
