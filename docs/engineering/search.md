# Search

## Purpose

Search enables users to discover information without knowing where it is stored.

Search is a cross-domain capability.

---

## Searchable Objects

- Courses
- Learning Assets
- Resources
- Tasks
- Assessments
- Questions
- Discussions
- Announcements
- Faculty
- Learners

---

## Search Principles

Results should prioritize relevance over recency.

Filters should be composable.

Permissions are always enforced before returning results.

---

## Metadata

Search indexes should include:

- Title
- Description
- Tags
- Author
- Institution
- Academic Context
- Last Updated
